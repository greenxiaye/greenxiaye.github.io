JS_map与forEach区别

map 与 forEach 区别

相同点

  1. 遍历数组
  2. 匿名函数中的 this 都是指 Window

不同点

  1. forEach 没有返回值，即返回值为 undefined

  2. map()返回一个新数组，原数组不会改变

    注意，如果 map 不写返回值，默认返回新的数组，每一项的元素是 undefined

  3. map 因为返回数组所以可以链式操作，forEach 不可以链式调用。

  4. map 速度比 forEach 快
