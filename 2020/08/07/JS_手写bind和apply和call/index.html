JS_手写bind和apply和call

call

  1. call 接受两个参数，第一个参数是 this 指向，第二个参数是函数接受的参数
  2. 改变 this 指向后，原函数会立即执行
  3. 此方法只是临时改变 this 指向一次

手写 call

call 简易核心版

call 好好写版

如果第一个参数传的是 null 或者 undefind，这个时候函数中的 this 会指向 window，所以需要进行一下判断，讲 context 指向 window

call 好好写版+独一无二的属性

apply

  1. apply 接受两个参数，第一个参数是 this 指向，第二个参数是函数接受的参数，以数组的形式传入
  2. 改变 this 指向后，原函数会立即执行
  3. 此方法只是临时改变 this 指向一次

手写 apply

注意，arguments 就是传进去的参数集合，通过 arguments[1]得到传进来的数组参数

apply 好好写版

apply 好好写版+边缘检测

  1. 如 Person.apply(12, [13, ‘男’])
    因为 12 不是对象，所以无法执行 context[fn]，因为没有 null[fn]
    所以不是对象的，比如 number，boolean，string，要 new 一下。
    注意，typeof 12 为 number，所以 new Number 一下
  2. 如 Person.apply(null, [13, ‘男’])
    如果第一个参数传的是 null 或者 undefind，这个时候函数中的 this 会指向 window，所以需要进行一下判断，讲 context 指向 window
    (context == null || context == undefined) && (context = window);
  3. 函数检测，this 如果不是构造函数，则抛出异常

bind

  1. bind 接受两个参数，第一个参数是 this 指向，第二个参数是函数接受的参数，这个参数列表可以分多次传入

  2. 改变 this 指向后不会立即执行

  3. 返回一个永久改变 this 指向的函数

  4. bind 返回一个新函数，执行新函数的的时候 this 指向 obj,然后把 bind 时传入的参数和调用新函数时传入的参数合并，一起传给新函数。
